# Vastu Shastra Website

A premium, highly interactive website for Vastu Shastra consultation and guidance.

## Tech Stack
- Frontend: React (Functional components, Hooks)
- Styling: Tailwind CSS
- Backend Readiness: Firebase service utilities (abstracted)
